# 英语阅读助手

一个本地运行的英语阅读辅助工具，用于把英文长难句拆成渐进式主干层级，并提供重点词组、单词词源与语义演变解释。

## 功能

- 按段落排版英文原文
- 点击单句后按需进行主干分层分析
- 高亮每一层新增的英文和中文信息块
- 提取重点词组并支持点击解释
- 支持单词/词组的词根词缀、原始含义、语义演变链和当前语境含义
- 本地缓存分析结果，减少重复 API 调用

## 本地运行

需要 Node.js 20 或更新版本。

1. 复制环境变量示例：

```bash
copy .env.example .env
```

2. 在 `.env` 中填写你的 OpenAI API Key：

```env
OPENAI_API_KEY=
```

3. 启动应用：

```bash
npm start
```

默认访问地址以 `.env` 中的 `PORT` 为准，例如：

```text
http://localhost:5187/
```

Windows 用户也可以双击 `start-app.cmd` 启动。

## 安全说明

`.env`、缓存、日志和本机快捷方式不会提交到 Git。请不要把真实 API Key 写入代码或 README。
